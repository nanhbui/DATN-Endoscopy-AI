# Implementation Plan: Video Source Selection Modal

**Branch**: `003-video-upload-modal` | **Date**: 2026-05-02 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/003-video-upload-modal/spec.md`

## Summary

Replace the three-tab toggle (Tải video / Trực tiếp / Thư viện) in the workspace with a two-control layout: a modal-trigger button (opens the source selection dialog) and a live-stream button. The modal combines upload-new and library-select into a single `VideoSourceModal` Dialog component. Zero backend changes — all APIs from spec 002 are reused as-is.
